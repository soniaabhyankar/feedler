Prerequisites

Modern Web Browser (eg. Google Chrome)
NodeJS (Download NodeJS from their official website)

Installations

Install Vue CLI
Open a terminal with administrative privileges

Run command:

npm install -g @vue/cli

Open a terminal in the project directory

Run command (may require administrative privileges):

npm install

Then, run command:

nodemon app

Keep this terminal open, and open another terminal in the project directory

Run command:

npm run serve

Keep this terminal running as well

Then, in the browser go to http://localhost:8080
